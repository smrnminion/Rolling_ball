Detailed Description

Класс для создания структуры "шара" для вычитания фона.

Этот класс создает структуру "шара", которая может быть использована для моделирования удаления фона из изображений. "Шар" представляет собой матрицу значений, имитирующую объемный шар, который катится под изображением для удаления фоновых элементов.
